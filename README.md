# 🛡️ 信息安全管理系统课程开源社区建设

## 📖 项目概述

本项目是《信息安全管理》课程的开源社区建设实践作业。我们旨在通过真实的开源协作模式，学习并应用信息安全管理的相关知识。项目采用模块化协作方式，由本课程小组成员共同维护、开发与测试。

## 📂 项目结构 

本仓库主要分为两大核心模块：

### 1. 🛡️ 漏洞靶场/

* **内容说明**：存放选定的漏洞靶场代码。
* **功能简述**：本项目集成了 **SQLI-LABS** 经典 SQL 注入挑战环境（Less 1-10）。
* **实验目的**：用于模拟真实的 SQL 注入场景，提供从报错注入到布尔/时间盲注的研究环境，帮助社区成员掌握 Web 漏洞的攻防原理。

### 2. 💻 成员代码/

存放各成员开发的个人原创工具。每个成员均有独立的代码目录，具体功能如下：

#### 👤 组长个人代码目录 - renyanbin/
* **功能简述**：
  本脚本实现了一个面向多模态大模型安全性的自动化测试与防御框架，其集成了 **AMIA (主动遮蔽意图分析)** 防御协议，通过 Patch 切片与动态掩码技术拦截针对 LLaVA 等模型的图像越狱攻击。代码针对科研实战环境进行了深度优化，支持数据集暴力读取、实时结果落盘（防崩溃丢失数据）以及灵活的显存调试逻辑，并能自动生成 DSR 防御成功率统计报告，实现了从底层防御触发到高层指标评估的闭环实验。

#### 👤 tangzekai 个人代码目录 - tangzekai/
* **功能简述**：LeetCode 每日一题爬虫程序：
1. 获取每日题目信息：通过调用 LeetCode 中国站（leetcode.cn）的 GraphQL API，查询当天的每日一题。
2. 获取题目详情：根据题目的 titleSlug 再次调用API，拉取题目的详细内容，包括中文/英文标题、难度、标签、中文/英文题面描述、提示（Hints）和示例测试用例。
3. 清洗 HTML：将题目描述中的 HTML 标签转换为可读性更好的纯文本Markdown 格式。
4. 输出与保存：
    - 在终端以易读格式打印题目信息
    - 输出完整的 JSON 数据
    - 将结果保存到以日期命名的 JSON 文件（如 daily_question_2026-02-09.json）

#### 👤 xiezhizhuo 个人代码目录 - xiezhizhuo/
* **功能简述**：基于 TCP 协议的文件下载客户端程序，可与服务器建立连接并根据用户输入的文件名请求下载文件，服务器返回文件内容后客户端将其写入本地完成下载；程序具备文件名合法性校验、完善的异常处理和传输统计功能，能统计接收字节数、耗时与传输速度。

#### 👤fengyongjia 个人代码目录 - fengyongjia/
* **功能简述**：一个改进的LSB图像隐写系统，主要功能包括：
1. 文本与二进制互转：text_to_binary 将字符串转为二进制串，binary_to_text 反之。
2. PSNR计算：calculate_psnr_grayscale 计算两幅灰度图像的峰值信噪比，用于评价隐写图像质量。
3. 改进LSB隐写（ImprovedLSB类）：
   隐藏消息时：将秘密文本转为二进制，在图像R通道的次低有效位（bit 1）中，通过固定随机种子生成随机位置依次嵌入，输出携密图像。
   提取消息时：根据相同的随机位置序列，从次低有效位读取二进制串并还原为文本。
4. 主程序演示：读取 buptgray.bmp，将字符串 "BUPTshahexiaoqu" 随机嵌入到次低有效位，生成 buptgraystego1.bmp，再提取并验证，同时计算并输出PSNR值。
整体上，相比传统LSB（固定最低位），该方法提高了隐蔽性（次低有效位+随机位置），但容量略减。

#### 👤weichunru 个人代码目录 - weichunru/
* **功能简述**：本工具用于将数字水印嵌入到载体图像中，采用分块DCT变换技术。主要功能包括：图像读取与通道处理、8×8分块DCT变换、基于系数大小关系的水印嵌入与提取、嵌入结果可视化对比显示、嵌入前后图像质量评估（PSNR）、提取水印相似度评估（NC）以及高斯噪声攻击测试。
